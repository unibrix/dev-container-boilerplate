# Slidev deck

Source deck for the Dev Containers demo presentation.

## Commands

From `docs/slidev/`:

```bash
npm install
npm run dev
```

Open `http://localhost:3030`.

If you run this inside the Dev Container, VS Code should auto-forward port `3030`.

Static build:

```bash
npm run build
```

The deck content is derived from [../CONTAINERS-DEMO.md](../CONTAINERS-DEMO.md) and reuses assets from `docs/`.
